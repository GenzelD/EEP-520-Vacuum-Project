## Agent: 'vacuum'
**Description** A robot agent that will collect dirt and avoid obstacles. It is seen as a green box with 3 sensors used to avoid the environment aroud it.
**Capabilites** Avoids walls and furniture and collects dirt agents in the arena.

## Agent: 'dirt'
**Description** A static agent that is represented as a brown box. The goal of the robot is to grab these particles. When the robot touches these agents, they will be destroyed.
**Capabilites** Static agents that will be collected by robot.
